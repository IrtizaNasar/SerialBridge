Make sure the Serial Bridge app is running!

How to use this example:

  • 1. Start the Serial Bridge desktop app
  • 2. Connect your Arduino Nano 33 BLE or Uno R4 WiFi
  • 3. Ensure the connection ID is set to device_1
  • 4. Upload ble-nano33.ino or ble-uno-r4.ino to your board
  • 5. Open this HTML file in your browser
  • 6. Use keys 1, 2, and Space to control the Arduino LED!

What this example does:

  • Connects to the Serial Bridge at http://localhost:3000
  • Sends logic commands to device_1 via Bluetooth
  • Receives sensor data from the Arduino
  • Visualizes data and connection status
