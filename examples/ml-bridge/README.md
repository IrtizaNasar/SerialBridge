# ML Bridge Examples

This folder contains Arduino sketches and examples specifically designed to work with **ML Bridge**.

## Contents

### 1. BLE LED Controller (`/ble-led-controller`)
A beginner-friendly example showing how to control an LED wirelessly using machine learning predictions.

- **Hardware:** Arduino Uno R4 WiFi (or any Arduino with BLE).
- **Concept:** ML Bridge sends a label (e.g., "class_1"), and the Arduino turns on an LED.
- **Protocol:** Bluetooth Low Energy (BLE).
