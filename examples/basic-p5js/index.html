Make sure the Serial Bridge app is running!

How to use this example:

  • 1. Start the Serial Bridge desktop app
  • 2. In the Bridge app, add a connection and select arduino_1 as the ID
  • 3. Connect your Arduino to a USB port
  • 4. Upload the Arduino sketch that sends sensor data (e.g., Serial.println(analogRead(A0));)
  • 5. Open this HTML file in your browser
  • 6. Watch the visualization respond to your Arduino data!

What this example does:

  • Connects to the Serial Bridge at http://localhost:3000
  • Listens for data from arduino_1
  • Visualizes the data as a bar chart and line graph
  • Shows connection status with a colored indicator
