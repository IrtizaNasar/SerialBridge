Usage Guide
P5

Connect your sketch

Include the socket.io client and our bridge script to start receiving data. 

<script src="http://localhost:3000/socket.io/socket.io.js"></script>
<script src="http://localhost:3000/serial-bridge.js"></script>

Use in P5.js

Initialize the bridge and listen for data: 

let bridge;

function setup() {
    createCanvas(400, 400);
    
    bridge = new SerialBridge();
    
    bridge.onData('device_1', (data) => {
        console.log('Received:', data);
    });
}

OSC Settings

Enable OSC Broadcasting (Send)

Send device data to a target IP address via OSC.

Data is sent to address /serial with arguments [DeviceID, Data].

Enable OSC Receiving (Listen)

Listen for incoming OSC commands from any source.

Listens on this port for commands sent to /send with arguments [DeviceID, Message]. No target IP needed - accepts from any source.
